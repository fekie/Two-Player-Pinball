# Score Counter

## Setup
Connect pins from the lcd to the arduino as follows:
```
GND -> GND (breadboard)
VCC -> 5V
SDA -> A4
SCL -> A5
```
Connect pins from player 1 button to arduino as follows:
```
+ -> GND (breadboard)
- -> GPIO 13
```
Connect pins from player 2 button to arduino as follows:
```
+ -> GND (breadboard)
- -> GPIO 13
```
